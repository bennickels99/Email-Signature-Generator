# Email-Signature-Generator
A basic HTML page with small JavaScript elements that will generate a signature based on user filled out fields.
There are 5 fields that a user can fill out. They can generate a preview of their signature before they open to send
an email. To send the email there is another button that takes the fields and formats the signature to be sent in an email. The purpose of this is for the user to send an email to themselves that they can save in their inbox to copy
and paste, or save in their Email app.
